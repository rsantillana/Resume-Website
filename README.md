# Renz Santillana Resume Website

A responsive personal resume and portfolio website prepared for GitHub Pages.

## Files

- `index.html` — main website
- `assets/Renz_Santillana_Resume.pdf` — downloadable resume

## Publish with GitHub Pages

1. Create or open the repository named `Resume-Website`.
2. Upload `index.html` and the `assets` folder to the top level of the repository.
3. Open **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`.
6. Click **Save**.
7. Open:
   `https://rsantillana.github.io/Resume-Website/`

## Before publishing

- Confirm your GitHub profile address in `index.html`.
- Add LinkedIn when available.
- Replace project descriptions or add repository links as your projects are published.
- Remove your phone number from the public website if you do not want it visible online.
